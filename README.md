# Feedback Intelligence 

## DB connector library for internal use

A comprehensive Python library providing robust, type-safe database connectors for various database systems. This
library simplifies database interactions while maintaining clean, efficient, and secure connections.

## Features

- **PostgreSQL Read-Only Operations**
    - Thread-safe connection handling
    - Session management
    - Dynamic model creation
    - Type-safe queries
    - Connection pooling support

## Requirements

### System Requirements

- Python 3.10 or higher
- PostgreSQL 9.6 or higher
- pip package manager
